# System Info Script

A Python script that displays basic system information from the command line.

## What it shows
- Current user and hostname
- Current working directory
- Disk usage (total, used, free) in GB
- Current date and time

## How to run it
```bash
python3 system_info.py
```

## What I learned
- Importing and using Python modules (os, socket, shutil, datetime)
- Reading system information using Python
- Formatting output with f-strings
- Converting bytes to gigabytes

## Tech used
- Python 3.12
- WSL (Ubuntu)
